# AutoEval - Demo Script for Hackathon Presentation

## 🎯 Elevator Pitch (30 seconds)

> "AutoEval is an AI-powered hackathon scoring platform that automatically evaluates project submissions using GPT-4. It analyzes code quality, documentation, creativity, and relevance to provide instant, objective scores and detailed feedback. No more manual judging - let AI do it in seconds!"

## 🚀 Live Demo (5 minutes)

### Part 1: Host Creates Hackathon (1 min)

**Say**: "I'm a hackathon organizer hosting an AI Innovation Challenge"

1. Navigate to **Host Dashboard**
2. Fill in quick form:
   - Name: "AI Innovation Challenge 2025"
   - Theme: "Build an AI tool to solve a real-world problem"
   - Evaluation Prompt: "Focus on innovation, technical implementation, and real-world impact"
3. Click **Create Hackathon**

**Say**: "That's it! Our hackathon is live and ready for submissions"

### Part 2: Participant Submits Project (2 min)

**Say**: "Now I'm a participant submitting my project"

1. Navigate to **Submit Project**
2. Select the hackathon
3. Enter team details:
   - Team: "Team Innovation"
   - Project: "Smart Task Prioritizer"
   - Description: "AI-powered tool that intelligently prioritizes your daily tasks"
4. Upload files (pre-prepared):
   - Python code file
   - README with documentation
5. Click **Submit**

**Say**: "Watch this - the AI is now analyzing our code and documentation in real-time"

### Part 3: View Results (2 min)

**Wait for results to load (~30 seconds)**

**Say**: "And here are the results!"

**Highlight**:
- Overall score prominently displayed
- Four criteria scored individually:
  * Relevance to theme
  * Technical complexity
  * Creativity/innovation
  * Documentation quality
- Detailed AI feedback explaining the scores
- Visual progress bars making it easy to see strengths and weaknesses

**Say**: "This detailed feedback helps participants understand exactly where they excelled and where they can improve"

### Part 4: Leaderboard (30 sec)

1. Click **View Leaderboard**

**Say**: "The leaderboard automatically ranks all submissions, with top 3 getting special recognition"

**Show**:
- Rankings with medals
- Scores comparison
- Quick access to view any submission

## ❓ Anticipated Questions & Answers

**Q: How accurate is the AI evaluation?**
A: GPT-4 is trained on millions of code examples. We've found it correlates strongly with expert human judges, with the added benefit of consistency and detailed explanations.

**Q: What about code that the AI hasn't seen before?**
A: GPT-4 understands programming concepts, not just syntax. It can evaluate novel approaches and creative solutions.

**Q: Can it be gamed?**
A: The AI looks at multiple factors - code quality, documentation, relevance. Simply adding verbose comments won't help if the core code is weak.

**Q: What's the cost?**
A: Using OpenAI API, each evaluation costs ~$0.02-0.05. For a 100-person hackathon, that's under $5 - much cheaper than judge compensation.

**Q: Does it replace human judges?**
A: It can, or it can serve as a first-pass filter, letting human judges focus on top submissions only.

**Q: What file types does it support?**
A: All major programming languages plus documentation formats. It can even handle .zip archives.

**Q: How long does evaluation take?**
A: Typically 30-60 seconds depending on submission size. Much faster than hours of manual review.

**Q: Is the data secure?**
A: Files are stored locally on your server. We can add encryption and access controls for production use.
