# Calculator Project

## Overview
A simple yet functional calculator implementation in Python, demonstrating clean code practices and proper documentation.

## Features
- Basic arithmetic operations (add, subtract, multiply, divide)
- Power/exponentiation function
- Error handling for division by zero
- Clean, object-oriented design

## Installation
No external dependencies required. Just run with Python 3.x:

```bash
python test_sample.py
```

## Usage

```python
from test_sample import Calculator

calc = Calculator()
result = calc.add(5, 3)
print(result)  # Output: 8
```

## Technical Details
- **Language**: Python 3
- **Design Pattern**: Object-Oriented
- **Testing**: Manual testing included in main()
